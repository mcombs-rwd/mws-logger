Applications may request historical rain data via the API.

/rain-api
GET— Entry to API
?meter=
GET— Test data includes: green_roof, rain_gauge
&resolution=
GET— May be one of: hourly, daily, monthly
&start=
GET— dd-mm-yy hh:mm, will be rounded to nearest period
&stop=
GET— dd-mm-yy hh:mm, will be rounded to nearest period, defaults to now.
&format=
GET— May be one of: json, xml, csv. Defaults to csv.

Examples

Rain Gauge, one day, by hour, as CSV

http://localhost:8000/rain-api?sensor=rain_gauge&resolution=hourly&start=2024-03-24&stop=2024-03-24&format=csv

Green Roof, one week, by day, as JSON

http://localhost:8000/rain-api?sensor=green_roof&resolution=daily&start=2024-03-24&stop=2024-03-30&format=json

Green Roof, three months, by month, as XML

http://localhost:8000/rain-api?sensor=green_roof&resolution=monthly&start=2024-02-01&stop=2024-04-30&format=xml
